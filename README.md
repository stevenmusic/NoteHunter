# 🎮 NoteHunter

NoteHunter is a mobile-first sight-reading RPG game built with HTML, CSS, and JavaScript.

## 🎼 Features

- Sight reading training (Treble & Bass Clef)
- C2–C6 note range support
- Combo system
- Score system
- RPG progression system (Level / HP / ATK / DEF)
- Mobile vertical UI design

## 🎮 Game Modes

- 🎼 Treble Clef (C4–C6)
- 🎼 Bass Clef (C2–C4)
- 🎼 Mixed Mode (future update)

## 👤 RPG System (Planned)

- Weapons (ATK boost)
- Armor (HP boost)
- Shields (DEF boost)
- Level up system
- Monster battles

## 🏆 Future Features

- Online leaderboard
- PvP battle mode
- Equipment system with images
- Daily challenges
- Boss fights

## 📱 Platform

- Mobile-first design (portrait mode)
- Runs on GitHub Pages

## 🚀 How to Run

Open `index.html` in browser or deploy via GitHub Pages.
