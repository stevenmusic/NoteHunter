# 🎮 NoteHunter

Mobile vertical sight-reading RPG game inspired by incremental RPG UI.

## Features
- Tap Titans style dark UI
- EXP + Level system
- Sight reading training (C–B notes)
- Combo system
- RPG character stats
- Equipment system

## Pages
- Home (HUD + EXP bar)
- Character (stats + equipment)
- Play (game modes)
- Rank (coming soon)
- Settings (coming soon)
