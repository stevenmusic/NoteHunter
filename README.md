# 🎮 NoteHunter

Mobile sight-reading RPG game.

## Features
- Mobile bottom navigation UI
- Sight reading (C–B notes)
- Combo system
- RPG character system
- Equipment system (weapon / armor / shield)

## Pages
- Home (level / exp)
- Character (stats + equipment)
- Play (modes)
- Rank (coming soon)
- Settings (coming soon)

## Game Modes
- Treble Clef
- Bass Clef
- Mixed Mode
