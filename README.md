# 🎮 NoteHunter

Mobile-first sight-reading RPG game.

## Features
- Sight reading training (C–B notes)
- Combo system
- Score system
- Mobile vertical UI

## Modes
- Treble Clef
- Bass Clef (coming soon)
- Mixed Mode (coming soon)

## Future
- RPG equipment system
- Monsters & battles
- Leaderboard
- Online PvP
